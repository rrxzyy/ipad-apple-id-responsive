# iPad Apple ID Responsive Website

my responsive website project clone of Apple's iPad page, built with HTML, Tailwind CSS, and JavaScript. It also integrates AOS (Animate On Scroll) library to for scrolling animations. This website was developed as part of the **Frontend Web Development: From Zero To Hero** course at **Fast Campus Global**.

## Mentor

This project was developed under the guidance of [Naufaldi](https://github.com/naufaldi).

## Tech Stack

- HTML
- Tailwind CSS
- JavaScript
- AOS
